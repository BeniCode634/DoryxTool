Transparência em Permissões

Saiba exatamente quais permissões são utilizadas e por qual motivo específico

Permissões Necessárias

  • Ver canais - Para visualizar onde criar tickets
  • Enviar mensagens - Comunicação nos tickets
  • Gerenciar threads - Para criar threads de tickets
  • Criar threads privadas - Tickets privados

Permissões Não Utilizadas

  • Banir membros - Sem funcionalidade de ban
  • Gerenciar cargos - Não altera cargos
  • Excluir servidor - Impossível para o bot
  • Gerenciar webhooks - Não utiliza webhooks

Arquitetura Segura

Desenvolvido com as melhores práticas de segurança para proteger seus dados

Armazenamento Local

Todos os dados são armazenados localmente no servidor, sem transferência para terceiros.

Privacidade Total

Nenhum dado pessoal é coletado ou armazenado, apenas informações essenciais.

Proteção de Dados

Sistema de criptografia para proteger configurações sensíveis e garantir integridade.
